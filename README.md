# Smart ClassRoom

A modern classroom management web app built with TanStack Start, React 19, Tailwind CSS v4, and Lovable Cloud (Supabase) for auth and data.

## Features

- Student management
- Class & captain feedback
- Seat planner
- School rules
- Notifications
- SOS alerts
- Activity logs
- AI Syllabus Summarizer (Lovable AI Gateway — Gemini)

## Tech Stack

- **Framework:** TanStack Start v1 (React 19, Vite 7)
- **Styling:** Tailwind CSS v4
- **Backend:** Lovable Cloud (Supabase — auth, database, RLS)
- **AI:** Lovable AI Gateway (Google Gemini)
- **Data:** TanStack Query
- **Icons:** Lucide React

## Development

```bash
bun install
bun run dev
```

App runs at `http://localhost:8080`.

## Deployment

Published via Lovable. Any push to the connected GitHub repo syncs back to Lovable automatically.
